# Sistema_De_Estacionamiento_Basic (Python)

Un programa en consola que simula la gestión de un estacionamiento: registro de entrada, salida, cálculo de tarifa y control de espacios disponibles.

## Funcionalidades actuales
- Registrar vehículos con nombre, cédula y hora de entrada.
- Buscar información de usuario mediante dato especifico 
- Calcular monto a pagar según el tiempo de estancia.
- Liberar espacio automáticamente al salir.
- Mostrar información confidencial (historial completo).
- Menú interactivo en consola.

## Librerías utilizadas
- `datetime` para manejo de fechas y horas.

## Posibles mejoras futuras
- Persistencia de datos con **JSON o base de datos SQL**.
- Control de usuarios (cliente / administrador), dividir menu en dos partes.
- Reporte de ingresos diarios.
- Mejorar validación de entradas.
- utilizar el limpiador de pantallas OS, para que se vea mejor visualmente en consola.
